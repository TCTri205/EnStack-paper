# Images Folder

Thư mục này chứa các hình ảnh và biểu đồ được sử dụng trong tài liệu dự án EnStack.

## Danh sách hình ảnh cần thiết

### 1. t-SNE Visualizations
- `tsne_codebert.png` - Visualization của không gian biểu diễn CodeBERT (baseline)
- `tsne_enstack.png` - Visualization của không gian biểu diễn EnStack ensemble

### 2. Architecture Diagrams  
- `architecture_overview.png` - Sơ đồ tổng quan kiến trúc EnStack framework
- `training_pipeline.png` - Luồng xử lý training pipeline

### 3. Results Charts
- `performance_comparison.png` - Biểu đồ so sánh hiệu năng các mô hình
- `ablation_study.png` - Kết quả ablation study

## Hướng dẫn thêm hình ảnh

1. Thêm file ảnh vào thư mục này
2. Sử dụng định dạng PNG hoặc JPG
3. Đặt tên file theo quy ước: `lowercase_with_underscores.png`
4. Tham chiếu trong markdown: `![Alt text](images/filename.png)`
